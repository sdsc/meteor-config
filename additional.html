Keeping the pi from sleeping

1. sudo nano /etc/lightdm/lightdm.conf 2. scroll down to [SeatDefaults] 3. add xserver-command=X -s 0 dpms (to the line underneath [SeatDefaults]) 4. save and exit file with ctrl-X and Y to save 5. sudo shutdown -r now (reboots the pi)
